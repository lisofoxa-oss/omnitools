# OmniTools 🔧

A comprehensive collection of browser-based developer utilities. All processing happens **entirely in your browser** — nothing is uploaded to any server.

**Live:** [lisofoxa-oss.github.io/omnitools](https://lisofoxa-oss.github.io/omnitools)

## Tools

| Tool | Description |
|------|-------------|
| 📋 JSON Formatter | Format, validate, and compress JSON |
| 🔐 Base64 | Encode/decode Base64 with file support |
| 🔗 URL Encoder/Decoder | Encode and decode URLs |
| 📊 Text Diff | Compare two texts side by side |
| 🔍 Regex Tester | Test regular expressions with match highlighting |
| #️⃣ Hash Generator | SHA-1, SHA-256, SHA-384, SHA-512 |
| 🔤 Case Converter | camelCase, snake_case, kebab-case, and more |
| ⏰ Timestamp Converter | Unix timestamp ↔ human-readable date |
| 📝 Lorem Ipsum | Generate placeholder text |

## Development

```bash
npm install
npm run dev     # dev server at http://localhost:5173
npm run build   # production build to dist/
npm test        # run tests
```

## Deployment

Push to `main` → auto-deploys to GitHub Pages via GitHub Actions.
